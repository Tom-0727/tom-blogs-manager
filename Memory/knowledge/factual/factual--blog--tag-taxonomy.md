## Core Content

### 分类（Categories）- 一级分类，每篇文章只归一个

| 分类 | 英文 | 适用范围 |
|-----|------|---------|
| ResearcherZero | ResearcherZero | ResearcherZero项目的所有相关文章 |
| Agent工程 | Agent Engineering | Agent系统设计、源码拆解、工程实践 |
| Agent产品 | Agent Product | AI工具介绍、产品方案、自动化系统 |
| 展示 | Demo | Agent生成的报告/作品展示 |

如果后续需要新分类，参考方案B的三条内容线可能增加：
- Industry Insight（行业洞察）

### 标签（Tags）- 多选，每篇2-5个

格式规则：小写英文、kebab-case

| 标签 | 含义 | 适用场景 |
|-----|------|---------|
| researcher-zero | ResearcherZero项目 | 所有RZ相关文章 |
| arch-design | 架构设计 | 系统设计类文章 |
| agent-engineering | Agent工程 | 工程实现、代码级内容 |
| agent-product | Agent产品 | 产品介绍、方案设计 |
| agent-architecture | Agent架构 | 架构层面的思考 |
| skills | Skills机制 | 涉及Agent Skill系统 |
| memory | 记忆机制 | Agent Memory相关 |
| context | 上下文 | Context Engineering相关 |
| learning | 学习机制 | Agent Learning相关 |
| claude-code | Claude Code | 涉及Claude Code的使用或集成 |
| human-in-the-loop | 人机协作 | 涉及人类参与的设计 |
| ai-tools | AI工具 | 工具推荐、工具评测 |
| automation | 自动化 | 自动化方案、工作流 |
| open-source | 开源 | 开源项目分析 |
| mechanism-interpret | 机制解读 | 源码/机制级拆解 |
| intro | 项目介绍 | 新项目/新员工介绍 |
| release | 版本发布 | 产品发布公告 |
| demo | 演示 | Agent能力展示 |

### 后续可能新增的标签（根据方案B内容线）
- industry-insight（行业洞察）
- tool-review（工具评测）
- case-study（案例复盘）
- ai-trend（AI趋势）

## Limitations

- 标签数量应保持精简，避免过度细分导致标签页内容稀疏
- 新增标签前先检查是否可以复用已有标签
